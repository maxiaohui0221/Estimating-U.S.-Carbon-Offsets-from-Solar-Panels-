## ğŸ›  Methodology
The project was conducted in **R**, involving data cleaning, rigorous statistical analysis, and variance estimation. The following sampling designs were implemented and compared:

1.  **Simple Random Sampling (SRS):** Baseline estimation.
2.  **Stratified Sampling:** Stratified by **State** to account for varying sunlight intensity and solar adoption rates across regions.
3.  **One-stage Cluster Sampling:** Treating states as clusters.
4.  **Two-stage Cluster Sampling:** Sub-sampling within selected state clusters.

## ğŸ“Š Key Results
* **Stratified Sampling** proved to be the most effective method, yielding the smallest Standard Error (SE) and the narrowest Confidence Interval compared to SRS and Cluster methods.
* **Why:** The analysis revealed significant variance *between* states (e.g., California vs. Delaware) regarding solar potential. Stratified sampling effectively captured this heterogeneity, preventing the overestimation bias observed in cluster sampling.
